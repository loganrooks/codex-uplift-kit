# codex-uplift late orchestration recovery package

This folder is a one-time recovery/organizer package for `codex-uplift-kit` after v0.2 implementation advanced while orchestration/delegation/worktree docs were missing from the planning suite.

Entry point:

```text
START_HERE_RECOVERY.md
```

Recommended placement inside the repo:

```text
.planning/initiatives/codex-uplift-current-design-suite/_late-orchestration-recovery/
```

The files under `MISSING_DOCS_TO_COPY/` should be copied or merged into the parent initiative root.

The package tells the orchestrator how to reconcile the current partial implementation state, update routing/index/state/roadmap/release docs, create retrospective `.codex-uplift/` artifacts, and rerun hardening checks without restarting v0.2.
